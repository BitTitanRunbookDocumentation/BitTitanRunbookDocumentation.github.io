# Office 365
## Removes Send On Behalf Permission
### SYNOPSIS
```
    Remove 'Send On Behalf' Permission
```
### DESCRIPTION
```
    Remove 'Send On Behalf' Permission for mailboxes, distribution groups and dynamic distribution groups
```
