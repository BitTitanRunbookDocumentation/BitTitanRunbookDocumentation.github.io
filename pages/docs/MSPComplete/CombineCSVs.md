# MSPComplete
## Combine CSVs
### SYNOPSIS
```
    Combines two CSVs into a single CSV
```
### DESCRIPTION
```
    Combines two CSVs into a single CSV.
    The resulting CSV contains all the columns which were present in either of the two original CSVs.
    The resulting CSV will contain the columns in sorted order.
```
